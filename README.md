---
## Controls

| Key | Action |
|-----|--------|
| `W A S D` | Fly |
| `Q / E` | Move down / up |
| `Scroll wheel` | Zoom |
| `Right mouse + drag` | Look around |
| `Left Shift` | Sprint |
| `Space` | Pause / unpause |
| `[` / `]` | Halve / double time scale |

---

## Building

You need Rust and the raylib system library installed.

Run from the project root so the `shaders/` folder is found.
